# Wine Quality Dataset

This folder contains the UCI Wine Quality dataset (red and/or white wine CSV files).

## Source

Dataset from the UCI Machine Learning Repository:  
https://archive.ics.uci.edu/ml/datasets/wine+quality

## Usage

- The CSV files inside `data/wine+quality/` are used directly by the training scripts in `src/`.
- No additional download is required.
- These files are small and included in the repository for reproducibility.
